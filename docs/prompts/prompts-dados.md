# Prompts para Dados

## Prompt: Análise inicial de dataset

### Prompt

"Analise este conjunto de dados e diga o que devo observar primeiro."

### Problema

Resposta superficial e pouco acionável.

### Refinamento

"Analise este conjunto de dados e aponte qualidade dos dados, colunas críticas, possíveis valores ausentes, inconsistências e próximos passos para análise exploratória."

### Resultado

Resposta mais estruturada e útil para preparação de dados.

### Aprendizado

Quanto mais explícita a expectativa de análise, melhor a orientação recebida.
