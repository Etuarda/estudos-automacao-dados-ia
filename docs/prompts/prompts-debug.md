# Prompts para Debug

## Prompt: Debug de código Python

### Prompt

"Explique o erro neste código e sugira correção: ..."

### Problema

Resposta superficial, sem explicar a causa.

### Refinamento

"Explique o erro neste código Python, identifique a causa raiz, mostre a correção passo a passo e sugira uma alternativa mais robusta."

### Resultado

Resposta mais detalhada, com melhor valor de aprendizado.

### Aprendizado

Ser específica sobre profundidade e formato faz a IA sair do genérico.
