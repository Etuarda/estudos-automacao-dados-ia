# Prompts para Backend

## Prompt: Explicação de conceito backend

### Prompt

"Explique [conceito] com foco em backend, incluindo definição, exemplo prático e erro comum."

### Problema

Quando o pedido é genérico, a resposta tende a ficar ampla e pouco aplicada.

### Refinamento

"Explique [conceito] com foco em backend usando Node.js, com exemplo simples, analogia curta e erro comum de iniciante."

### Resultado

Resposta mais útil para estudo e revisão prática.

### Aprendizado

Especificar stack, nível e formato melhora bastante a qualidade.
