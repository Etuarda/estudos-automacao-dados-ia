# Glossário Geral

## IA

Uso de sistemas computacionais para executar tarefas associadas à inteligência humana.

## LLM

Modelo de linguagem treinado em grandes volumes de texto para interpretar e gerar linguagem natural.

## Prompt

Instrução dada ao modelo para orientar a resposta.

## Contexto

Informação adicional que ajuda o modelo a responder com mais precisão.

## Alucinação

Resposta incorreta apresentada com aparência de confiança.

## Token

Unidade de texto processada pelo modelo.

## Refinamento iterativo

Processo de melhorar um prompt a partir da análise da resposta anterior.
