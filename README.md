# Afya Data & AI Notes

Repositório organizado para registrar minha trajetória de estudos no bootcamp, reunindo anotações técnicas, exercícios, projetos, experimentos com IA e aprendizados aplicados ao longo dos módulos.

## Objetivo do repositório

Este espaço foi estruturado para funcionar ao mesmo tempo como:

- trilha de estudos documentada;
- base de revisão técnica;
- portfólio de evolução prática;
- registro do uso estratégico de IA no aprendizado.

Em vez de guardar apenas resumos soltos, a proposta é consolidar:

- anotações com explicação técnica em linguagem própria;
- exemplos de código e aplicações reais;
- exercícios resolvidos com contexto e aprendizado;
- projetos por módulo;
- prompts testados, refinamentos e troubleshooting;
- insights sobre backend, dados e IA.

## Como cada módulo será registrado

Cada módulo deve priorizar três frentes:

1. anotação técnica, sem copiar aula;
2. código ou exemplo prático;
3. aprendizado aplicado.

Estrutura esperada por módulo:

```text
modules/02-python-fundamentos/
└── notes.md
```

Modelo-base:

```md
# Nome do módulo

## Conceitos principais

## Como funciona

## Exemplo prático

## Aplicação real

## Erros comuns

## Insight
```

## Como exercícios e desafios serão documentados

Exercícios não ficam como código solto. Cada item deve ter contexto, solução e aprendizado.

Estrutura sugerida:

```text
exercises/python/exercicio-01/
├── solution.py
└── README.md
```

README do exercício:

```md
## Problema

## Abordagem

## Complexidade

## Aprendizado
```

## Parte estratégica do portfólio

Este repositório destaca a combinação de:

- Python aplicado;
- SQL aplicado;
- automação;
- backend orientado a dados;
- uso inteligente de IA para estudar, depurar e acelerar desenvolvimento.

## Ordem estratégica de estudo

A sequência abaixo orienta a evolução do portfólio de forma mais coerente:

1. Python fundamentos
2. Estruturas de dados
3. SQL
4. Automação com Python
5. POO
6. Preparação de dados
7. Machine Learning
8. IA generativa e assistentes
