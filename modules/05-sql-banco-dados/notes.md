# SQL e Banco de Dados

## Conceitos principais

- tabelas
- consultas
- filtros
- joins
- agregações

## Como funciona

Explique com suas palavras como os dados são organizados em tabelas e como as consultas recuperam informações úteis.

## Exemplo prático

```sql
SELECT nome, idade
FROM pacientes
WHERE idade > 18;
```

## Aplicação real

- relatórios
- busca de registros
- cruzamento de dados entre tabelas

## Erros comuns

- esquecer condição de join
- filtrar no lugar errado
- confundir `WHERE` com `HAVING`
