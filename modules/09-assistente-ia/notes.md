# Assistente de IA

## Conceitos principais

- prompt
- contexto
- memória
- instrução de sistema
- casos de uso

## Como funciona

Explique como um assistente combina instruções, contexto e regras de comportamento para responder melhor a tarefas específicas.

## Exemplo prático

```python
mensagens = [
    {"role": "system", "content": "Você é um assistente técnico."},
    {"role": "user", "content": "Explique APIs REST com foco em backend."}
]
```

## Aplicação real

- assistente de estudos
- suporte interno
- help desk técnico
- automação de respostas

## Erros comuns

- instruções vagas
- contexto insuficiente
- ausência de validação humana
