# Estruturas de Dados em Python

## Conceitos principais

- listas
- tuplas
- dicionários
- conjuntos
- iteração

## Como funciona

Explique aqui quando usar cada estrutura, quais diferenças importam e quais trade-offs aparecem no código.

## Exemplo prático

```python
usuarios = {"ana": 24, "bia": 29}

for nome, idade in usuarios.items():
    print(nome, idade)
```

## Aplicação real

- organização de coleções de dados
- agrupamento de informações
- preparação de estruturas para APIs e automações

## Erros comuns

- usar lista quando dicionário seria mais adequado
- mutabilidade inesperada
- percorrer estruturas sem entender o formato
